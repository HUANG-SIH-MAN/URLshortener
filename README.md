## 短網址產生器

#### 功能

- 使用者指可以輸入網址，並產生一組短網址
- 當使用者在瀏覽器輸入短網址，可以連接到原本網址的網頁
- 當使用者輸入相同的網址，會產生一組相同的短網址

---

#### 安裝與執行腳本

- 安裝
  - 下載專案
  ```
  https://github.com/HUANG-SIH-MAN/URLshortener.git
  ```
  - 安裝專案
    ```
    $ cd URLshortener
    $ npm install
    ```
  - 執行程式
    ```
    $ npm run start
    $ npm run dev
    ```
  - 伺服器位置
    ```
    localhost:3000
    ```
